# Tastenkürzel-Referenz

Zeigt alle Agent-Bridge und Cursor-native Tastenkürzel.

## Bridge Extension

| Shortcut | Aktion |
|----------|--------|
| `Ctrl+Alt+A` | Agent-Chat öffnen |
| `Ctrl+Alt+P` | Prompt-Dialog |
| `Ctrl+Alt+Shift+A` | Auswahl an Agent |
| `Ctrl+Alt+.` | Generierung stoppen |
| `Ctrl+Alt+K` | Diese Referenz |

## Cursor IDE

| Shortcut | Aktion |
|----------|--------|
| `Ctrl+I` | Composer |
| `Ctrl+Shift+I` | Composer Vollbild |
| `Ctrl+L` | Chat |
| `Ctrl+K` | Inline Edit |
| `Shift+Tab` | Modus: Agent → Plan → Ask |
| `Ctrl+Enter` | Senden |

## CLI

```bash
bridge shortcuts    # Vollständige Liste im Terminal
agent --help        # Cursor CLI Hilfe
```
