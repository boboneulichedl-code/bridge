### B.4 Ableitung `poc-v1-ui-modules.json` (Review, nicht erstellt)

**Fazit: ableitbar mit Einschränkungen.**

| Kategorie | Promotion-Inhalt |
|-----------|------------------|
| 1:1 übernehmen | `moduleId`, `version` (nach Freigabe semver), `supportedActions`, `requiredPermissions`, `supportedDomains`, `implementationPhase` → entfernen oder `active: true` |
| TBD belassen bis Designrules | `requiredDesignTokens`, `requiredComponents`, alle `*State.spec` |
| Nach P0.1 | `rollbackUndoVisibility` auf true erlauben wo Restore existiert |
| Nicht in Runtime ohne Gate | `proposalOnly`, `runtimeActive`, `designGapNotes` |
| Spec-Zusatzfelder | `emittedEvents`, `confirmationRules`, `mobileLayoutRules` — Spec PoC v1.1.0 verlangt diese |

**Blocker vor Ableitung:** DEC-P2-001 (Designrules-Pfad), DEC-P2-002 (D6 Render-Blocker), DEC-P2-003 (UI Permission Gate).
